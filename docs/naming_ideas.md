# Project Naming Ideas

## Requirements
- Works for current educational animation videos (Manim/3B1B style)
- Also works for future content types: stick figure motion graphics, 3D mannequin scenes, realistic character scenarios
- Not tied to "math" or any single content type
- Short, memorable, domain-available (ideally)

## Top Candidates

### Tier 1 (Recommended)

| Name | Meaning | Why |
|------|---------|-----|
| **Lumira** | "lumi" (light/illuminate) + "ra" | Premium feel, unique, works globally, scales to any content type |
| **MindReel** | mind + reel of video | Playful, immediately communicates "videos that teach", memorable |
| **Clareo** | From "clarity" | Elegant, the product is about making things clear, scales well |

### Tier 2 (Strong Options)

| Name | Meaning |
|------|---------|
| Vizora | Visual + explore |
| EduViz | Educational + visual (current working name) |
| Eureka | The "aha moment" — built into the video planner |
| Explainly | Dead simple, says what it does |
| Framewise | Frame by frame + wise/knowledge |

### Tier 3 (More Ideas)

| Name | Meaning |
|------|---------|
| Synthex | Synthesized explanations |
| LearnFrame | Each frame teaches you something |
| MindReel | Mind + reel |

## Decision
**Lumira** — chosen for its warm, illuminating quality that fits the educational + minimalist aesthetic. Tagline: "Understand anything visually."
